Flappy with Items

Like Flappy Bird Game





Flappy with Items is similar with Flappy game. However this game has different points with Normal Flappy game. That is additional Items. By this things, be more fun!




How to Play

Only push Space-key like matching with timing. Let's avoid pipes that coming towards.



How to install

You can use the script.

curl -sL "http://jusco.infogr.kushiro-ct.ac.jp/~p220032/script/install.sh | sh
